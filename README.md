# launcher
ссылка на игру для лаунчера: https://drive.google.com/file/d/19-aY3q4u8vdZo_kA7mgYFd2qrHbEfO_i/view?usp=sharing

для того, чтобы приложение запускало игру нужно распаковать архив в папку Debug

launcher\launcher\bin\Debug\

ВСЕ файлы игры закинуть просто в папку, НЕ НУЖНО создавать отдельную папку для игры, иначе программа не найдет файл с игрой
